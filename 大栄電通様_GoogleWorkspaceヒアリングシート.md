## 5. 改善したい業務

該当するものにチェック。顧客に自由回答を求めるより、近いものを選んでもらう。

- ファイル管理を整理したい
- メールアドレスを整備したい
- 社外から資料を見たい
- 会議後の議事録や共有を楽にしたい
- 営業報告を簡単にしたい
- 紙やExcelの管理を減らしたい
- AIを業務で使いたい
- 顧客管理をしたい
- 請求書発行や定型業務を効率化したい
- 社内マニュアルやナレッジを整理したい
- 営業資料・採用資料・Webサイトで魅力が伝わっていない

## 6. 課題カード

商談では、以下から「一番近いもの」を選んでもらう。

| 課題カード | 確認すること |
|---|---|
| 探す時間が多い | 資料、メール、過去データを探す時間が多くないか |
| 転記が多い | 紙、Excel、別システムへの二重入力がないか |
| 報告が遅い | 日報、現場報告、会議後共有が遅れていないか |
| 承認が止まる | 申請、見積、稟議が人待ちで止まっていないか |
| 情報共有が属人的 | 担当者しか知らない情報が多くないか |
| 紙やExcelが多い | 管理表、台帳、チェック表が散らばっていないか |
| AI活用が進まない | 使いたいが、社内データやルールが整っていないか |
| 魅力が伝わらない | 営業資料、採用資料、Webサイトに違和感がないか |

## 9. 注意事項

- kintone導入済み顧客には、Google Workspaceとの直接比較を避ける。
- 他社導入済みGoogle Workspaceは、顧客側が管理者権限を持っていればサポート可能。
- ライセンス費用は、原則として顧客がGoogleと直接契約する。
- 新規導入支援は、アカウント数・メール移行量・既存環境で金額が変わる。
